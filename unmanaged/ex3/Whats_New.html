HPE Synergy Software Releases - What's New?

★ HPE Synergy Composer 5.20.01 / HPE Synergy Image Streamer 5.20.01

A new HPE Synergy Management combination is available. HPE Synergy Composer 5.20.01 and HPE Synergy Image Streamer 5.20.01 offers new enhancements, and provides fixes for HPE Synergy components.

IMPORTANT:

Before performing an HPE OneView update, Hewlett Packard Enterprise recommends you use the HPE OneView Update Readiness Checker. The Update Readiness Checker is provided for HPE OneView users to evaluate the health of their HPE OneView appliance for a set of known conditions before proceeding with an appliance update. To download the checker, go to http://www.hpe.com/support/ov-urc.

HPE Synergy Composer 5.2 enhancements:

  • HPE Synergy integration with Cisco ACI

  • HPE Synergy static IP Addressing

  • IGMP Multicast enhancements

HPE Synergy Image Streamer 5.2 enhancements:

  • Enhanced storage management and the latest security updates

  • New OS Sample artifacts published for the following:

    • VMware ESXi 6.5 U3

    • VMware ESXi 6.7 U3

    • Red Hat Enterprise Linux 7.7

    • Red Hat Enterprise Linux 8.0

    • SUSE Linux Enterprise Server 12 SP5

    • SUSE Linux Enterprise Server 15

    • SUSE Linux Enterprise Server 15 SP1

For a complete list of fixes for HPE Synergy Composer, see the HPE OneView 5.2 and 5.20.01 Release Notes for HPE Synergy.

For a complete list of fixes for HPE Synergy Image Streamer, see the HPE Synergy Image Streamer 5.2 Release Notes.

★ HPE Synergy Custom SPP 2020.05.01 - May 2020

HPE Synergy Custom SPP 2020.05.01 is built on HPE Synergy component firmware.

IMPORTANT:

HPE Synergy Custom SPP 2020.05.01 contains new firmware versions for the HPE Virtual Connect SE 40Gb F8 Module and the HPE Virtual Connect SE 100Gb F32 module. On systems running HPE OneView 5.0 (or later), firmware versions for these Virtual Connect modules prior to those contained in this HPE Synergy Custom SPP may result in unexpected behavior.



For more information, see Customer Advisory a00099807.



For a complete list of fixes, see the HPE Virtual Connect SE 40Gb F8 Module for HPE Synergy Release Notes and the HPE Virtual Connect SE 100Gb F32 Module for HPE Synergy Release Notes.

HPE Synergy Custom SPP 2020.05.01 includes firmware/driver updates for:

  • Gen9/Gen10 System ROM

  • Gen10 Smart Array Controllers

    This combination is vSAN certified with firmware version 2.65 and smartpqi driver 1.0.4.3008-1.

  • Gen9 Smart Array Controllers

    This combination is vSAN certified with firmware version 7.00 and nhpsa driver 2.0.44-1.

  • Integrated Smart Update Tool/Smart Update Manager

    For more information, see Hewlett Packard Enterprise Information Library.
