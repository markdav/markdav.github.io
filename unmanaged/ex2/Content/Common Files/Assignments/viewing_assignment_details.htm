Viewing Assignment Details

To view the details of a role, complete the following steps:

  1. On the CCS console home page, under the QuickStart menu, click Manage.

  2. Click the Roles tab. The Roles Management page displays the CCS predefined and custom roles.

  3. Select one of the roles to display the role details. The ROLE page shows the following information:

  • App Name - Name of the app.

  • App Slug – App API parameter

  • App ID – App unique identifier number

  • Description – App description

  • Created by – Name of the company who created the app

  • Docs – This is the URL where you can get more information about the app

  • Email Notification – The individual receiving email inquires, requests

  • Deployed regions – The regions where this app is installed.   To see region details, click the ellipsis (…) next to a region.

    • Region – Region where the app is active

    • Status – Active/Not-active

    • Action Required – Apps waiting for approval
