Creating Assignments

To create assignments, complete the following steps. 

  1. On the HPE GreenLake CCS console home page, under the QuickStart menu, click Manage.

  2. Click the Assignments tile.

  3. Click Create Assignment link. The Create Assignment window displays.

  4. Select the user you want to create an assignment. Begin by typing a user's name into the Added User(s) search field. If a user is registered with CCS, their name is displayed in the search field.  

  5. Click the Add button to add roles to the user. The application AddRoles pane displays.

Figure 1. Add Roles Dialog

 

  1. Select the application and role you want to assign this user. To view the roles you selected, click the ViewSelectedRoles link at the bottom of the pane. Next, you can optionally define a scope for each Role. A scope can be a set of dynamic resources that you assign a specific user. Think of scope as a way of assigning additional responsibilities to the user beyond the original roles' primary duties.

  2. Click the +AddScope to define a scope for the role you selected. The Scopes pane displays.

    Figure 2. Assignment Scopes Dialog

     

  3. Select the scopes you wish to add and click the Add button. To view the scopes you selected, click the ViewSelectedScopes link at the bottom of the pane.

  4. Click CreateAssignment. The assignments page displays the user and assignments.
