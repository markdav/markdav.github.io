HPE Common Cloud Services (CCS) Console

The HPE Common Cloud (CCS) Console is a cloud-based application that provides a set of services for HPE applications that CCS users can consume and interact with from a single location. 

About the HPE Common Cloud Services Console

To understand how to effectively use the features and functionality available through the CCS console, review the following topics:

  • Navigating the HPE Common Cloud Services Console

  • Common Cloud Services QuickStart Menu

Navigating the HPE Common Cloud Services Console

HPE Common Cloud Services is a cloud-native solution offering network and application management in one easy-to-use console.

  • CCS QuickStart Icon - When you click this () icon, the quick-access menu displays. From this menu, users can access provisioned apps, access the app catalog, and manage user roles and assignments.
  • App Catalog – Access the entire CCS application catalog. From this page, you can provision and launch apps
  • Invite Users – You can ask other team members to join CCS by clicking the Send Invite button. See Inviting Users for details.
  • SSO/SAML Connection – Administrators can set up SAML and SSO for their business unit and customers Help Icon – The help() icon navigates to the available CCS help content. Home Icon - The home (  )icon returns to the CCS console home page.

 

Common Cloud Services Console QuickStart Menu

The Common Cloud Services QuickStart menu allows you to navigate to different sections of the CCS application quickly. The QuickStart menu, located on the CCS console's left side, displays when you click the icon. The QuickStart menu is available from all CCS console screens. When not in use, the QuickStart menu remains hidden on the left side of the CCS console.

Inviting Users

A Common Cloud Services administrator can add users to CCS (team members) by inviting them to join CCS. Use the following steps to invite users to join CCS.

  1. Navigate to the HPE Cloud Console Services home page, and click the Send an Invite link.

  2. Enter the user's email address.

  3. Click Enter.

An email is sent to the user, inviting them to sign up for a Common Cloud Account. After users receive the invitation, they are redirected to the Common Cloud Services signup page.

Company Account Details

Use the account details page to modify and make changes to your general company account.
