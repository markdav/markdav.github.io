Users Account

The Users account is where group membership and roles are managed. The following activity can be applied to Users account.

  • Searching for Users
  • Using Search Filters
  • Inviting Users
  • Deleting Users

Searching for Users

Use the search option to find other HPE GreenLake CCS group members. When you enter a name in the search field, the names of team member populate the display. The information returned for users include their email address and their current status which can be verified or unverified.

Using Search Filters

You can filter your search for users based on their verification status.

  1. Click the filter icon .
  2. In the filter dialog select Verified or Unverified.
  3. Click Apply Filter.

Inviting Users

Administrators can easily invite team members to join Common Cloud Services. Use the following steps to invite users to CCS.

  1. Click the Invite Users link.
  2. Enter the user's emailaddress.
  3. Click Send Invite.

An email is sent to the user, inviting them to sign up for a HPE GreenLake Common Cloud Account. After users receive the invitation, they are redirected to the HPE GreenLake Common Cloud Services signup page.

Deleting Users

Use the following steps to delete users.

  1. Select the User.
  2. Click Delete Users link.
  3. Confirm User account deletion. Click Delete User.
