Navigating the HPE GreenLake CCS User Interface

When you first log in to the HPE GreenLake CCS portal, you see the main CCS console. Select one of the CCS services available. On the left side of the screen there is a default QuickStart menu you can access when you click the icon. The default QuickStart menu displays on the left-side of the console and allows you to easily navigate HPE GreenLake CCS.

The default QuickStart menu lists the HPE GreenLake CCS activities you can perform.

Home

Use the Home link to navigate back the HPE GreenLake Common Cloud Services console. You can use this link to return the console from any page within HPE GreenLake CCS.

My Apps

My apps page list all the applications available to launch. You can add apps your account using the app onboarding process. Apps that you successfully onboard and provision will show on your My Apps page.

App Catalog

The App Catalog is where all HPE GreenLake Cloud application available to CCS are listed. The process of adding apps to the catalog is managed through the HPE Common Cloud Services.

Manage

The Manage Account page allows you configure edit your account profile and other customer account services. HPE GreenLake Cloud CCS user can manage the following services.

  • Account Details page allows you to edit your account details. Account details your can modify include company name, address, and country.
  • Users can look up HPE GreenLake CCS users and find out what roles they have from the Users page. You also invite users using the link at the top of the page.
  • Roles is where you manage and assign roles to HPE GreenLake CCS users. Roles can be assigned from CCS pre-built roles or you can create your own custom roles.
  • Assignments page is where you create assignments and to give roles to users.
  • Device Management page allows user to fully manage and provision devices.
